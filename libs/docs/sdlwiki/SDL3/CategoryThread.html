Introduction

This category contains functions for system independent thread management routines.

{i} ''NOTE'': You should not expect to be able to create a window, render, or receive events on any thread other than the main one. For platform-specific exceptions or complicated options ask on the [https://discourse.libsdl.org/ forums/mailing list].
